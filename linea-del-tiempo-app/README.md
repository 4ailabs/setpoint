# Linea del Tiempo (React)

App React creada con Vite.

## Requisitos
- Node.js 18+ (recomendado)

## Desarrollo
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy en Vercel
- Importa el repo en Vercel.
- Framework preset: **Vite**.
- Build command: `npm run build`.
- Output directory: `dist`.

## Estructura
- `src/App.jsx`: Tabs principales y contenido.
- `src/styles.css`: Estilos base.
- `index.html`: Carga de Mermaid.
